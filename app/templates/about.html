About SmartQuery

A natural language to SQL agent built on top of the AdventureWorks 2016 dataset. Ask questions in plain English — the system generates, validates, and executes T-SQL queries automatically.

How it works — Agent Pipeline

01 Intent Classifies if the query is a valid database request or off-topic.
02 Generator Produces a T-SQL SELECT query using the dynamic schema context.
03 Guard Blocks unsafe operations and known hallucinated columns.
04 Executor Runs the validated query directly against SQL Server.
05 Explainer Summarises the query logic in plain English for the user.

Tech Stack

Backend
FastAPI
REST API with async endpoints and Jinja2 templates.
LLM
Groq · Llama 3.1
Fast inference for intent detection, SQL generation, and explanation.
Database
SQL Server
AdventureWorks 2016 via pyodbc with ODBC Driver 17.
Safety
SQL Guard
Rule-based layer blocking destructive ops and hallucinated columns.
Schema
Dynamic Loader
Keyword-aware schema filtering to stay within LLM token limits.
Frontend
Vanilla JS
No framework. Particle system, SQL syntax highlighting, live pipeline status.
